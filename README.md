## Project Overview
This project focuses on analyzing customer transaction data using **Microsoft Excel** to generate meaningful business insights. The goal is to transform raw transactional data into an **interactive and visually appealing dashboard** that supports data-driven decision-making.

The project demonstrates end-to-end data analytics skills including **data cleaning, analysis, visualization, and dashboard creation**.

## Objectives
- Clean and prepare raw transaction data
- Analyze customer behavior and sales performance
- Identify trends across regions, products, and customer segments
- Build an interactive Excel dashboard for easy insights
- Present business insights in a simple and visual format

## Dataset Description
The dataset contains customer transaction records with the following columns:

- Transaction_ID  
- Date  
- Customer_ID  
- Customer_Name  
- Product_ID  
- Product_Name  
- Category  
- Quantity  
- Unit_Price  
- Payment_Method  
- Region  
- Customer_Segment  
- Customer_Since  
- Total_Amount
